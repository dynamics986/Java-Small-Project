## About
This repository contains a compact Java project, which is intended as a small, self-contained example or utility. The code is designed to be easy to read and modify, making it suitable as a learning resource or as a starting point for a larger project.

## Prerequisites
- Java JDK 8+ (or the specific Java version you used; e.g., Java 11, Java 22).
- Netbeans APP (IDE22+)

## Structure

### 1. Application_Demo
A comprehensive learning project with demonstrations of core Java concepts:

- **Array_Demo** - Arrays and data structure fundamentals
- **Exception_Demo** - Exception handling and error management
- **FileDownloader** - File I/O and network operations
- **FileIO_Demo** - File input/output operations
- **GUI_Demo** - Graphical User Interface development
- **Inheritance_Demo** - Object-oriented inheritance concepts
- **Loop_Demo** - Loop structures and iterations
- **Polymorphism_Demo** - Polymorphic design patterns
- **Recursion** - Recursive algorithms and function calls

### 2. Assignment_Demo
A collection of assignment-based projects demonstrating practical applications:

- **BoardGame** - Board game implementation
- **ChocoFactory** - Factory pattern demonstration
- **DataProcessor** - Data processing and manipulation
- **HKWeatherReport** - Weather data reporting system
- **NewPiFormula** - Pi calculation using modern algorithms
- **OldPiFormula** - Pi calculation using classical algorithms
- **Stego** - Steganography implementation

### 3. Francis_Project
A complete NetBeans IDE project with structured build system:

- **src/** - Source code directory
- **build/** - Compiled class files
- **dist/** - Distribution package
- **build.xml** - Apache Ant build script
- **nbproject/** - NetBeans configuration files
- **manifest.mf** - JAR manifest file


```
Java-Small-Project/
├── Application_Demo/          # Java fundamentals demonstrations
├── Assignment_Demo/           # Practical assignment projects
└── Francis_Project/           # Complete NetBeans project
    ├── src/                   # Source files
    ├── build/                 # Compiled files
    ├── dist/                  # Distribution
    ├── build.xml              # Build configuration
    ├── manifest.mf            # JAR manifest
    └── nbproject/             # NetBeans project files
```
